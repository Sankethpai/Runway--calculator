# Runway Calculator

## Project Overview
Runway Calculator is a web-based application designed to assist in calculating various runway-related metrics, providing accurate and reliable calculations for aviation professionals.

## Design Choices
- **User-Centric Design**: The interface is designed for ease of use, allowing users to quickly access the features they need.
- **Responsive Layout**: The application is responsive and works on various devices, ensuring accessibility for all users.
- **Modular Architecture**: The application is built using a modular architecture, allowing for easy updates and maintenance.

## Features
- **Runway Length Calculation**: Calculate necessary runway length based on aircraft type, weight, and environmental factors.
- **Weight Capacity Metrics**: Understand the weight limits for various aircraft on different runway conditions.
- **Performance Data**: Access performance data for various aircraft, including takeoff and landing speeds.

## Tech Stack
- **Front-End**: HTML, CSS, JavaScript (React framework)
- **Back-End**: Node.js, Express.js
- **Database**: MongoDB
- **Hosting**: Deployed on Heroku

## Installation
To run the project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Sankethpai/Runway--calculator.git
   cd Runway--calculator
   ```
2. Install the necessary packages:
   ```bash
   npm install
   ```
3. Start the local server:
   ```bash
   npm start
   ```
4. Open your browser and navigate to `http://localhost:3000`.

## Usage Instructions
1. Enter the aircraft specifications (weight, type, etc.) into the input fields.
2. Choose the environmental conditions affecting the runway (humidity, temperature, elevation).
3. Click on the ‘Calculate’ button to retrieve the necessary runway length and other metrics.
4. Review the results displayed on the screen.
